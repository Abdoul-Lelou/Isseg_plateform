# ISSEG Platform Monorepo

Monorepo pour la plateforme ISSEG (Institut Supérieur des Sciences de l'Éducation de Guinée).

## Structure
- `apps/api`: Backend NestJS
- `apps/web`: Frontend React (Vite)
- `apps/worker`: Worker asynchrone Node.js (BullMQ)
- `services/moodle-service`: Micro-service de synchronisation Moodle
- `packages/shared`: Code partagé (Types, DTOs, etc.)
- `packages/ui`: Composants React partagés
- `packages/config`: Configurations communes
